請根據精簡後的"環境資訊"與"agent旅行記憶"整理資料，與"agent profile資料"去生成各個 agent 的 decision-making 結果。

active mode包含五種如下，請依據各個agent不同的性質、旅行記憶與環境資訊，將這五種mode去填入active mode值:
"fast"                 // 想要快一點
"tolerate_congestion"  // 繼續塞車也沒關係
"avoid_congestion"     // 避開壅塞
"comfortable"          // 穩定舒適
"short_distance"       // 想走短一點

agent name為每個agent的名字，請根據"agent profile資料"的name值去對應生成，不許有漏掉。
vehicle_type根據"agent profile資料"的"vehicle_ownership"值去對應生成，不許有漏掉。
decision-making 結果輸出必須完全符合以下結構，除此結構內容外，禁止產生其餘文字內容：
{
  "agents": [
    {
      "agent name": "王大偉",
      "active mode": "fast",
      "residential_location": "東區",
      "vehicle_type":"機車"
    }
  ]
}
