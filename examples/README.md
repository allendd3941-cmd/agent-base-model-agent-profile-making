# Examples

This directory contains small, curated examples for GitHub review.

## Files

- `gama_request_init.json`: Example initialization payload sent from GAMA.
- `gama_request_step.json`: Example step-update payload sent from GAMA.
- `sample_outputs/`: Selected LLM outputs copied from local runs.

The full local `output/` directory is intentionally ignored by Git because it contains generated runtime artifacts.
